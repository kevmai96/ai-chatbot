# ai-chatbot

# What it does:
1. Builds a Rule-Based chatbot (pattern matching)
2. Builds an ML-Based chatbot (trained on examples)
3. Compares both approaches side by side
4. Runs an interactive chat session

# How to run:
1. Setup environment
python3 -m venv env
source env/bin/activate

2. Install requirements:
pip install -r requirements.txt

3. Run: python app.py
